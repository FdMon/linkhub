# LinkHub

A Linktree clone built with React, Vite, and Tailwind CSS.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

## Development

Start the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

Build for production:

```bash
npm run build
```
